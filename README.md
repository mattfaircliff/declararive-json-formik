# Formik forms using declarative JSON configs

This project was created by Matthew Faircliff as a learning tool to accompany the Medium article at https://medium.com/@mfaircliff.

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn build`

Builds the app for production to the `build` folder.\
